# Agent Templates Example

Loads every public agent template, validates required fields, and prints a short gallery index.

This example is mock-only. It does not send live payments, store keys, or call private backend internals.

```bash
npm run examples:templates
```

Expected output:

```txt
Loaded 11 DNA x402 agent templates
- prediction-market-agent [prediction-market]
- solana-trading-agent [solana-trading]
...
```
