# Parad0x Signal Reseller Example

Shows how a builder can package a Parad0x signal into a paid agent response while preserving attribution, the Parad0x source fee, the builder fee, and the DNA 0.1% rail fee.

```bash
npm run examples:paradox-signal-reseller
```

Expected output includes a signal usage event, visible fee lines, and receipt metadata.

No real signal feed keys, wallet keys, or live payments are used.
