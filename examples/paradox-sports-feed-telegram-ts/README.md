# Parad0x Sports Feed Telegram Example

Shows a Telegram signal bot flow using Parad0x sports signals as the source alpha.

The example proves:

- Telegram delivery creates a signal usage event.
- Parad0x Labs attribution stays visible.
- Signal source fee, builder fee, and DNA 0.1% fee are present.
- No bot-held private keys or live payments are used.

```bash
npm run examples:paradox-sports-telegram
```
