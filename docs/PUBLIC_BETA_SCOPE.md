# Public Beta Scope

Public Beta is open for builders and users to ship agent/API commerce on the hosted DNA x402 rail.

Open:

- create agents
- paper agents
- public/private profiles
- copy settings
- builder APIs
- paid data feeds
- paid tools
- low-risk capped live payments

Live payment requirements:

- Solana USDC
- client-side signing
- per-transaction and daily caps
- provider proof
- DNA 0.1% treasury proof
- receipt verification

Outside Public Beta scope:

- backend custody
- backend signing
- hidden fees
- auto-sweep
- unrestricted autonomous live trading

Coming after separate safety gates:

- additional settlement networks
- direct builder fee collection
- unrestricted public copy trading
- regulated/high-risk verticals
- physical goods
- public netting
