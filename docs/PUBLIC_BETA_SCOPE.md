# Public Beta Scope

Public Beta is open for builders and users to test agent/API commerce with hard safety rails.

Open:

- create agents
- paper agents
- public/private profiles
- copy settings
- builder APIs
- paid data feeds
- paid tools
- low-risk capped live payments

Live payment requirements:

- Solana USDC
- client-side signing
- per-transaction and daily caps
- provider proof
- DNA 10 bps treasury proof
- receipt verification

Outside beta scope:

- backend custody
- backend signing
- hidden fees
- auto-sweep
- unrestricted autonomous live trading
- physical goods
- high-risk categories
- public netting
- broad multi-chain production settlement
