# Demo Backtest Agents

Demo agents are allowed, but fake PnL is not.

Demo/backtest output must be clearly marked:

```txt
SIMULATED / DEMO
```

Show:

- strategy description
- sample trades
- sample PnL
- sample size
- caveat
- receipt-gated report if paid

Do not claim demo data is live proof.

[`demo-proven-edge-agent.json`](../templates/agents/degen-live/demo-proven-edge-agent.json)
