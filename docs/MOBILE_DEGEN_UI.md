# Mobile Degen UI

The mobile Degen surface should be fast and blunt:

- active agents
- open positions
- live/paper PnL
- receipts
- copy links
- referral stats
- max pain
- pause button

Do not hide the risk profile behind settings. The user should see the ape budget, daily max pain, open exposure, and kill switch before any live intent.
