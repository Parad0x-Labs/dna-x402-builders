# One-Click Agent Mode

One-Click Agent Mode is the fast path:

```txt
Connect wallet. Pick agent. Set max pain. Launch.
```

The builder should show a simple preview before saving:

- agent type
- wallet model
- mode
- venue
- ape budget
- daily max pain
- open exposure
- slippage
- pause rule
- fees
- receipt behavior

Live actions are trade intents first. The user still signs on their wallet/client. DNA x402 handles quote, payment proof, receipt, fee waterfall, paid unlock, and proof trail.
