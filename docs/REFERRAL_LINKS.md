# Referral Links

Degen Mode referral links let builders share agents, rooms, signals, APIs, and profiles.

Targets:

- copy agent
- join room
- buy signal
- use API
- follow profile

Reward modes:

- fee share
- builder fee share
- source fee share
- none

Default rule:

```txt
Referral earns from visible fees, not follower losses.
```

Loss-rake defaults are outside Public Beta scope and require separate review.
