# Live PnL And Leaderboards

Leaderboards must make fake flexing harder.

Show:

- PnL
- ROI
- win rate
- average entry
- average exit
- max drawdown
- volume
- follower profit
- copy count
- live/paper badge
- receipt proof count
- sample-size badge

Warnings:

- `HIGH_AVG_ENTRY`
- `LOW_SAMPLE_SIZE`
- `HIGH_DRAWDOWN`
- `PAPER_ONLY`

A 95% win rate with 96c average entry is not the same thing as edge. Show average entry and sample size.
