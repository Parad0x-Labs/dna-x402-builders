## Builder Flow

Pick it. Fork it. Prompt it. Ship it. Get paid.

1. Open a template JSON.
2. Copy the `suggestedPrompt`.
3. Point Cursor or another coding agent at this repo.
4. Build against the hosted API URL in `DNA_X402_API_URL`.
5. Keep examples mock by default until your receipt and fee flow is wired.

## Safety Defaults

These templates are built around the hosted DNA x402 rail:

- visible fee waterfalls
- signed receipts
- replay-safe paid retry
- receipt-bound builder fees
- receipt-bound Parad0x signal source fees where a Parad0x feed is used
- client-side wallet ownership
- no backend custody
- no backend signing
- no hidden fees
